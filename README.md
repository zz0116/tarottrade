## Deploy to GitHub Pages

```bash
# Create repo on GitHub named "tarottrade"
# Then:
git remote add origin https://github.com/zz0116/tarottrade.git
git branch -M main
git push -u origin main

# Then in GitHub → Settings → Pages → deploy from main branch / (root)
```
